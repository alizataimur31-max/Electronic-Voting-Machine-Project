# Electronic Voting Machine

This Python project simulates an electronic voting machine using Tkinter.

## Features
- GUI for voter information input
- Input validation (name, phone number, CNIC, province, rural/urban)
- Voting window with selectable political parties (Pakistan Based)
- Saves voter information and votes to text files
- Reset functionality

## How to Run
1. Make sure Python is installed.
2. Run the script: `python electronic_voting_machine.py`
3. Follow GUI instructions to enter info and cast votes.
